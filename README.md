# ClearCare Kenya - Insurance Intelligence Platform

A user-friendly insurance management system for Kenyan healthcare providers.

## Features
- Patient registration
- Insurance coverage verification (NHIF, Jubilee, AAR, etc.)
- Pre-authorization requests
- Patient referrals
- Find in-network hospitals
- AI assistant for help

## Getting Started

### Prerequisites
- Any web browser (Chrome, Firefox, Edge)

### Running the System
1. Open `index.html` in your web browser
2. That's it! No installation required.

## Working Together

### Using Git for Collaboration

**1. One person creates a GitHub repository:**
```bash
# Create account at github.com
# Click "New repository"
# Name it "clearcare-system"
# Copy the repository URL
```

**2. Connect to GitHub and push:**
```bash
git remote add origin https://github.com/YOUR-USERNAME/clearcare-system.git
git branch -M main
git push -u origin main
```

**3. Your friend clones the repository:**
```bash
git clone https://github.com/YOUR-USERNAME/clearcare-system.git
cd clearcare-system
```

**4. Daily workflow:**
```bash
# Before starting work
git pull

# After making changes
git add .
git commit -m "Describe your changes"
git push

# If there's a merge conflict, talk to your friend and resolve it together
```
